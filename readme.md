### Description
A small clone of Excel written in TypeScript

### What is used
The project used TypeScript, Jest, Webpack, Commitizen, Scss

### Functional
What is currently available

* You can use different functions, for example, =ABS(A1). It is also possible to specify a range (A1:A5) and handle nested functions (=ABS(СУММ(A1:A5)))
* Copy, paste, change cell color and background using context menu
* Change cell sizes
* Saving data to the site's local storage

### Run project
Before running, make sure you have NodeJs and TypeScript installed

To run the application, run the following commands:

Installing packages
```cmd
npm i
```

Starting a local server
```cmd
npm run start
```
